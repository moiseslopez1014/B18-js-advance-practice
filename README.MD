# Advance Practice JavaScript

This is the evaluable project at final module for JavaScript.

We will use an arrange of set of tools, skills, and knowledge gained through the module to elaborate a fictional WebApp for films.

With the help of TMDB's API, this project will have access to a database from tmdb.org, to transform this data and show it categorized a sorted, to offer visionate of this movies's details, from the poster and title, to the year, casting and crew members.

## Sections

This project consist in a single website that dinamically change its contents to show two main sections that switch with the client's behavior.

#### Nav Bar
With a logo, navigation buttons and a search input

#### Content Bar
With buttons to navigate, switch the view style and a selector for categories.

#### Main section
Where the movies will show, it will refresh dinamically to offer the content in different views, show a different list of movies, or showing the detail of a particular movie. 
This is the part were we have to develop completely.

#### Footer

As usual. header and footer is provided by the teacher, so we focus in the JS part of the project.
